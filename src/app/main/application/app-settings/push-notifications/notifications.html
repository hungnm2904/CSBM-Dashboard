Android Push Notification

Android push notidication requires GCM sender ID and GCM API key to enable

GCM sender ID

To get your GCM sender ID, enable GCM for your Android project in the Google Developer Console. Take note of your project number. It should be a large integer like 123427208255. This project number is your GCM sender ID.

GCM API key

To get your GCM API key, go to the Google developer credentials page, and either create a new API key (server type) or reuse an existing one.
